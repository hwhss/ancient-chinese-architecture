# 2026-03-26 添加古建筑地图浏览功能

## 功能说明
新增「古建筑导览地图」页面，用户无需使用AI对话，可直接浏览所有古建筑信息。

## 主要改动

### 1. 新增地图页面 (frontend/pages/map/map.vue)
- **分类浏览**：皇宫、桥梁、园林、城防、民居、楼阁、水利
- **建筑卡片**：展示图片、名称、位置、简介、标签
- **16个古建筑**：涵盖 README 中提到的代表性建筑
- **一键跳转**：点击卡片进入详情页查看大图
- **返回AI**：底部按钮可返回AI聊天页面

### 2. 更新首页 (frontend/pages/index/index.vue)
- 添加快捷入口按钮「🗺️ 浏览古建筑地图」
- 欢迎消息提示可直接点击地图入口
- 扩展关键词映射支持全部20个古建筑

### 3. 更新详情页 (frontend/pages/detail/detail.vue)
- 扩展素材名称映射支持新增的建筑
- 支持从地图页传入建筑名称参数
- 优化图片加载逻辑

### 4. 更新路由配置 (frontend/pages.json)
- 将地图页设为首页（第一个页面）
- 保留AI聊天页和详情页

## 使用方式

1. 打开应用默认进入「古建筑导览地图」页面
2. 点击顶部分类标签筛选建筑类型
3. 点击建筑卡片进入详情页查看大图
4. 点击底部「💬 询问AI」按钮进入AI聊天界面
5. 在AI聊天页可点击「🗺️ 浏览古建筑地图」返回地图

## 技术实现

- 使用 `scroll-view` 实现横向分类标签滚动
- 使用 `v-for` 渲染建筑卡片列表
- 使用 `computed` 属性实现分类筛选
- 使用 `uni.navigateTo` 进行页面跳转
