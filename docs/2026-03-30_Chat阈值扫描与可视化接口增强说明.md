## 2. Chat 阈值扫描（新增）

## 2.1 新增脚本

- 文件：`backend/scripts/scan-chat-thresholds.js`
- npm 命令：`npm run scan:chat-thresholds`

功能：

1. 自动遍历 `CHAT_VECTOR_SIMILARITY_THRESHOLD` 与 `CHAT_HYBRID_SCORE_THRESHOLD` 的小网格。
2. 对错别字、别名、口语、无关问题做批量回归。
3. 输出每组阈值通过率、误召回数，并给出最优候选。

## 2.2 推荐配套参数

当前已在默认配置中对齐：

1. `CHAT_VECTOR_SIMILARITY_THRESHOLD=0.20`
2. `CHAT_HYBRID_SCORE_THRESHOLD=0.20`
3. `CHAT_VECTOR_MIN_DOMAIN_SIMILARITY=0.45`（新增，低相似度防误召回）

说明：

- 当关键词完全未命中且向量相似度低于该门槛时，会回退 AI，避免无关问题误命中知识库。

## 3. 可视化接口增强（按 temp.md 对齐）

## 3.1 保留原接口

1. `GET /api/visualization/overview`
2. `GET /api/visualization/map-points`
3. `GET /api/visualization/timeline`
4. `GET /api/visualization/stats`

## 3.2 新增兼容别名

1. `GET /api/visual/info-graphic` -> 复用 timeline
2. `GET /api/visual/chart` -> 复用 stats

## 3.3 字段增强摘要

### overview

新增：

1. `type: "overview"`
2. `kpis[]`（开发看板可直接渲染）

### map-points

新增：

1. `type: "map-points"`
2. `markers`（`points` 的别名）
3. `geoJson`（FeatureCollection，可直接对接地图组件）
4. `points[].model3d`（`status/viewerType/modelUrl/poster`）

### timeline

新增：

1. `type: "timeline"`
2. `infographic`（`nodes/edges/labels/links`，用于信息图渲染）

### stats

新增：

1. `type: "chart"`
2. `chart`（ECharts 友好结构：`title/unit/xAxis/series/legend/tooltip`）
