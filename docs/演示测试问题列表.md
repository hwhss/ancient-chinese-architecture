## 测试检查要点

### 1. 查看响应中的 debug 信息
```json
{
  "source": "knowledge_base" | "ai",
  "matchedBy": "exact_match" | "keyword_match" | "vector_match" | "hybrid_match" | "ai_fallback",
  "vectorRetrievalEnabled": true,
  "retrievalTopCandidates": [...]
}
```

### 2. 验证向量检索是否生效
- `vectorRetrievalEnabled` 应为 `true`
- `retrievalTopCandidates` 应包含候选问题及相似度分数

### 3. 验证匹配来源
- `source: "knowledge_base"` - 命中知识库，节省 Token
- `source: "ai"` - 未命中，调用 LLM 生成
