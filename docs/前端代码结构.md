# 前端代码结构树形图

> 按图索骥，快速定位代码功能

```
frontend/
├── 📄 入口与配置
│   ├── main.js                 # 应用入口：初始化Vue、全局挂载、路由拦截
│   ├── App.vue                 # 根组件：全局样式变量、主题色定义、页面容器
│   ├── manifest.json           # UniApp应用配置：应用名称、图标、权限
│   ├── pages.json              # 页面路由与导航：页面列表、TabBar配置、导航栏样式
│   ├── pages.config.js         # 页面编译配置
│   └── vue.config.js           # Vue/Webpack配置
│
├── 📱 页面 (pages/)
│   ├── home/home.vue           # 首页：建筑推荐、分类入口、每日一建
│   ├── map/map.vue             # 地图页：腾讯地图、建筑点位、筛选列表
│   ├── index/index.vue         # AI聊天页：对话界面、智能问答
│   ├── detail/detail.vue       # 详情页：建筑详情、素材展示、图文介绍
│   ├── viewer/viewer.vue       # 3D查看器：Three.js模型展示、LOD切换
│   ├── favorites/favorites.vue # 收藏页：我的收藏列表
│   ├── settings/settings.vue   # 设置页：应用设置
│   └── dev-settings/dev-settings.vue # 开发设置：API地址、调试选项
│
├── 🧩 组件 (components/)
│   ├── AppNavigation.vue       # 底部导航栏：Tab切换、图标状态
│   │
│   ├── chat/                   # 聊天相关组件
│   │   ├── ChatInputArea.vue   # 聊天输入框：消息输入、发送按钮
│   │   ├── ChatSidebar.vue     # 聊天侧边栏：历史会话列表
│   │   ├── SearchResults.vue   # 搜索结果展示：知识卡片、推荐问题
│   │   └── WelcomeHero.vue     # 欢迎区域：引导语、快捷入口
│   │
│   ├── home/                   # 首页相关组件
│   │   ├── HeroSection.vue     # 首屏区域：标题、背景、入口按钮
│   │   ├── DailyBuilding.vue   # 每日推荐：今日建筑卡片
│   │   ├── BuildingCard.vue    # 建筑卡片：缩略图、名称、朝代
│   │   ├── FeatureCard.vue     # 特色卡片：功能入口
│   │   ├── FeaturesSection.vue # 特色区域：功能列表
│   │   ├── KnowledgeSection.vue# 知识区域：知识条目展示
│   │   └── PreviewSection.vue  # 预览区域：内容预览
│   │
│   ├── map/                    # 地图相关组件
│   │   ├── TencentMapContainer.vue  # 腾讯地图容器：地图初始化、点位标记
│   │   ├── BuildingList.vue    # 建筑列表：侧边列表、筛选结果
│   │   ├── MapFilter.vue       # 地图筛选：朝代、类型筛选器
│   │   ├── MapHeader.vue       # 地图头部：标题、搜索
│   │   └── MapCharts.vue       # 地图图表：统计图表
│   │
│   ├── detail/                 # 详情页相关组件
│   │   ├── DetailHeader.vue    # 详情头部：建筑名称、返回按钮
│   │   ├── DetailContent.vue   # 详情内容：图文介绍
│   │   ├── DetailGallery.vue   # 详情画廊：图片轮播
│   │   └── DetailTabs.vue      # 详情标签：信息/素材/历史切换
│   │
│   ├── ui/                     # UI通用组件
│   │   ├── SectionHeader.vue   # 区块标题：带装饰线的标题
│   │   └── ActionButton.vue    # 操作按钮：主题样式按钮
│   │
│   ├── shared/                 # 共享组件
│   │   └── TraditionalIcon.vue # 传统图标：SVG图标库（宫殿/园林/桥梁等）
│   │
│   └── 独立功能组件
│       ├── DigitalSeal.vue     # 数字印章：传统印章效果
│       ├── InfoGraphic.vue     # 信息图形：结构图、关系图
│       ├── VisualChart.vue     # 可视化图表：数据图表展示
│       ├── LottieAnimation.vue # Lottie动画：MG动画播放
│       ├── OptimizedImage.vue  # 优化图片：懒加载、占位图
│       ├── ShareCard.vue       # 分享卡片：生成分享图
│       ├── SkeletonScreen.vue  # 骨架屏：加载占位
│       └── VirtualMessageList.vue # 虚拟列表：长列表优化
│
├── 🔌 服务层 (services/)
│   ├── api.js                  # API封装：请求拦截、错误处理、基础URL配置
│   ├── apiWithCache.js         # 带缓存的API：请求结果缓存
│   └── model3dTokenManager.js  # 3D模型令牌：模型访问权限管理
│
├── 🛠️ 工具函数 (utils/)
│   ├── cache.js                # 缓存工具：本地存储封装
│   ├── httpCache.js            # HTTP缓存：请求级缓存
│   ├── lazyLoad.js             # 懒加载：图片/组件懒加载
│   ├── performance.js          # 性能监控：性能指标收集
│   ├── mapPerformance.js       # 地图性能：地图渲染优化
│   ├── navigation.js           # 导航工具：页面跳转封装
│   ├── fontLoader.js           # 字体加载：动态字体加载
│   ├── fontSizeManager.js      # 字体大小：全局字号管理
│   ├── markdown.js             # Markdown解析：富文本渲染
│   ├── collectionManager.js    # 收藏管理：收藏增删查
│   └── animationHelper.js      # 动画辅助：动画工具函数
│
├── 🎨 样式 (styles/)
│   ├── design-tokens.css       # 设计令牌：CSS变量、主题色
│   ├── button-interactions.css # 按钮交互：点击效果、状态样式
│   └── hero-hover-effects.css  # 首屏悬停：悬停动画效果
│
└── 📦 静态资源 (static/)
    ├── tabbar/                 # TabBar图标：首页/名录/AI/收藏
    │   ├── home.png
    │   ├── home-active.png
    │   ├── map.png
    │   ├── map-active.png
    │   ├── chat.png
    │   ├── chat-active.png
    │   ├── favorite.png
    │   └── favorite-active.png
    └── draco/
        └── draco_decoder.js    # Draco解码器：3D模型压缩解码
```

## 功能对应速查

| 功能 | 入口页面 | 核心组件 | 服务/工具 |
|------|----------|----------|-----------|
| AI对话 | index/index.vue | ChatInputArea, SearchResults | api.js |
| 地图浏览 | map/map.vue | TencentMapContainer, BuildingList | - |
| 建筑详情 | detail/detail.vue | DetailHeader, DetailGallery, DetailTabs | api.js |
| 3D模型 | viewer/viewer.vue | - | model3dTokenManager.js |
| 收藏功能 | favorites/favorites.vue | - | collectionManager.js |
| 主题样式 | App.vue | - | design-tokens.css |
| 图片优化 | - | OptimizedImage | lazyLoad.js |
| 性能优化 | - | VirtualMessageList, SkeletonScreen | performance.js |
