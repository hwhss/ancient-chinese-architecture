## 一、项目概述

**项目名称**: 古建筑AI导览系统  
**技术栈**: Node.js + Express (后端) + UniApp/Vue3 (前端)  
**数据存储**: JSON文件 / PostgreSQL (可切换)  
**核心功能**: AI问答、3D模型浏览、建筑知识库、可视化展示

## 四、核心功能流程

### 4.1 AI问答流程

```
用户提问 → chatController → chatService
  → 问题归一化 (normalizeQuestion)
  → 混合检索 (hybridRetriever: 关键词+向量)
  → 知识匹配 → 答案重写 (rewriteKnowledgeAnswer)
  → 返回AI回答
```

### 4.2 3D模型加载流程

```
viewer.vue → getBuildingModel3dManifest
  → buildingService: 验证权限 → 签名URL生成
  → 返回manifest: 模型URL、LOD层级、热点、相机参数
  → Three.js加载 → DRACOLoader解压 → EffectComposer后期处理
```

### 4.3 数据流

```
前端请求 → Controller → Service → Repository
  → JSON文件 或 PostgreSQL查询
  → 返回格式化数据
```

## 五、技术特点

### 5.1 双数据源架构

- 通过 `DATA_SOURCE` 环境变量切换
- 开发时使用JSON，生产使用PostgreSQL
- 同一套Repository接口，无缝切换

### 5.2 资源安全

- 3D模型URL使用HMAC签名
- 支持过期时间、IP绑定、角色权限
- Token自动续期机制

### 5.3 前端CDN加载

- G6、Lottie、ECharts、Three.js均通过CDN加载
- 统一的 `loadScriptOnce` 机制避免重复加载
- 适配UniApp多端（H5、小程序、App）

### 5.4 混合检索

- 关键词匹配 + 向量相似度
- 可配置权重和阈值
- 支持问题归一化和答案重写

## 八、数据模型

### 8.1 建筑基础信息 (Building)

```javascript
{
  id: String,           // 建筑ID
  name: String,         // 名称
  category: String,     // 类别
  location: String,     // 位置
  province: String,     // 省份
  city: String,         // 城市
  lat: Number,          // 纬度
  lng: Number,          // 经度
  heritageLevel: String,// 遗产级别
  openStatus: String,   // 开放状态
  mainEraStart: Number, // 起始年代
  mainEraEnd: Number,   // 结束年代
  description: String,  // 描述
  image: String,        // 图片URL
  tags: Array           // 标签
}
```

### 8.2 建筑档案 (BuildingProfile)

```javascript
{
  id: String,                    // 建筑ID
  era: String,                   // 建造时代
  style: String,                 // 建筑风格
  overviewSummary: String,       // 概述
  keyPoints: Array,              // 关键特点
  history: String,               // 历史背景
  architectureHighlights: Array, // 建筑亮点
  culturalValue: String,         // 文化价值
  model3d: {                     // 3D模型配置
    status: String,              // 状态(none/planned/demo/ready)
    viewerType: String,          // 查看器类型
    poster: String,              // 封面图
    versions: Array              // 版本列表
  }
}
```

### 8.3 3D模型版本 (ModelVersion)

```javascript
{
  version: String,      // 版本号
  label: String,        // 版本标签
  format: String,       // 格式(glb/gltf)
  modelUrl: String,     // 模型URL
  draco: Boolean,       // 是否DRACO压缩
  ktx2: Boolean,        // 是否KTX2压缩
  lod: Array,           // LOD层级
  hotspots: Array,      // 热点标注
  camera: Object        // 相机配置
}
```
