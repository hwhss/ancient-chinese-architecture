## API路由速查表

| 端点 | 方法 | 控制器 | 功能说明 |
|------|------|--------|----------|
| /api/health | GET | healthController | 健康检查 |
| /api/chat | POST | chatController | AI问答 |
| /api/buildings | GET | buildingController | 建筑列表 |
| /api/buildings/:id | GET | buildingController | 建筑详情 |
| /api/buildings/:id/model3d | GET | buildingController | 3D模型数据 |
| /api/material | GET | materialController | 素材查询（单条） |
| /api/materials | GET | materialController | 素材列表 |
| /api/knowledge | GET | knowledgeController | 知识检索 |
| /api/visualization/overview | GET | visualizationController | 概览统计 |
| /api/visualization/map-points | GET | visualizationController | 地图点位 |
| /api/visualization/timeline | GET | visualizationController | 时间线 |
| /api/visualization/stats | GET | visualizationController | 统计数据 |
| /api/assets/signed | GET | assetController | 获取签名URL |
| /api/assets/refresh | GET | assetController | 刷新签名URL |
| /api/qiniu/download-url | GET | qiniuController | 七牛下载链接 |
| /api/qiniu/object | GET | qiniuController | 七牛对象读取 |
| /api/qiniu/image | GET | qiniuController | 七牛图片获取 |

## 核心流程对应

| 功能流程 | 入口 | 核心服务 | 数据层 |
|----------|------|----------|--------|
| AI问答 | chatController | chatService → hybridRetriever | dataRepository |
| 建筑查询 | buildingController | buildingService | dataRepository |
| 3D模型 | buildingController | buildingService | data-jsondb/ |
| 资源签名 | assetController | qiniuService/signedAsset | - |
| 向量检索 | - | vectorSearchService | PostgreSQL/pgvector |
| 关键词检索 | - | keywordRetriever | data-jsondb/ |
